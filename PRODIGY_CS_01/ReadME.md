## Implement Caesar Cipher

Description
The Caesar Cipher is one of the simplest and most widely known encryption techniques. This project implements a basic Caesar Cipher in Python, allowing you to encrypt and decrypt messages by shifting letters in the alphabet.

Usage

## 1.Clone the repository

`git clone https://github.com/Kingisline01/PRODIGY-Intenship_Repo.git`

`cd Caesar cipher`

## 2.Run the script:

`python main.py`

## 3.Follow the on-screen instructions to encrypt or decrypt a message.

Enter your message: `Hello, World!`

Enter shift value: `3`

Encrypted message: Khoor, Zruog!
