# Simple Keylogger

Description

This project implements a simple keylogger in Python. The keylogger records keystrokes and saves them to a file. Use responsibly and only on systems you own or have permission to monitor.

## Usage

## Clone the repository:

`git clone https://github.com/Kingisline01/PRODIGY-Intenship_Repo.git`

`cd Simple Keylogger`

## 2.Run the script:

`python main.py`

## 3.The keylogger will start recording keystrokes and save them to a file.

Keylogger is running... Press Ctrl+C to stop.
Keystrokes saved to keylog.txt
